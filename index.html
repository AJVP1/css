CSS (Cascading Style Sheets)

Fundamentos

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilos utilizado para describir la presentación de un documento HTML. Permite aplicar estilos a elementos HTML, como colores, fuentes, márgenes y diseño en general.

Nota: Para poder agregar estilos en el archivo HTML es necesario en el head agregar, <link rel= "stylesheet" href = "style.css">.

Maquetación

Una declaración de CSS es un bloque que especifica el conjunto de estilos que se añadirán a un elemento HTML. Su estructura contiene lo siguiente:

  • Selector: define el elemento o conjunto de elementos a los cuales se añadirán los estilos
  • Propiedad: es el nombre del estilo de CSS
  • Valor: es el valor asignado a la propiedad
CSS
h1 {
   color: blue;
}

En este ejemplo, el h1 es el selector, el color sería la propiedad y el valor sería blue.

Selectores basicos

Los selectores básicos en CSS son:

  • Selector de tipo: selecciona elementos por su nombre de etiqueta (p, h1, div, etc.)
  • Selector de clase: selecciona elementos por su atributo class ( .mi-clase)
  • Selector de ID: selecciona un elemento único por su atributo id ( #mi-id)
  • Selector de atributo: selecciona elementos por un atributo específico ( a[href="ejemplo"])
  • Selector universal: selecciona todos los elementos ( *)

Selectores combinadores

Un selector combinador es la combinación de dos o más selectores para seleccionar elementos que cumplen con múltiples criterios. Los combinadores más comunes son:

  • Combinador descendente (espacio): selecciona elementos que son descendientes de otro elemento (ejemplo: div p)
  • Combinador hijo (>): selecciona elementos que son hijos directos de otro elemento (ejemplo: div > p)
  • Combinador adyacente (+): selecciona un elemento que es inmediatamente siguiente a otro (ejemplo: h1 + p)
  • Combinador general (~): selecciona elementos que son hermanos de otro elemento (ejemplo: h1 ~ p)

Especificidad

En algún punto, cuando estés creando una página web, te encontrarás con problemas con los estilos, por ejemplo: ¿Por qué no se aplica el color que le estoy poniendo? ¿Por qué este elemento se comporta de manera diferente? Probablemente, sea un inconveniente de cascada o especificidad

La cascada es el concepto que determina qué estilos se colocan sobre otros, priorizando a aquellos que se encuentren más abajo del código. Recordarás que CSS es la abreviación de Cascade Style Sheets, que traducido es hojas de estilos en Cascada. Por lo tanto, si al principio del código decimos que los h1 van a tener un cierto color y después decimos que van a tener otro color, entonces tendrán el último color que se les asigne.

La especificidad consiste en dar un valor a una regla CSS sobre qué tan específico es el estilo, esto para que los navegadores puedan saber qué estilos aplicar sobre otros, independientemente de dónde se encuentren en el código. El estilo se aplicará donde la especificidad sea mayor

Tipos de especificidad:

  • Especificidad de tipo: 1 punto
  • Especificidad de clase: 10 puntos
  • Especificidad de ID: 100 puntos
  • Especificidad en línea: 1000 puntos
  • Especificidad !important: 10000 puntos

Pseudoclases

Una pseudoclase define el estilo de un estado especial de un elemento. la sintaxis es la siguiente:

CSS
selector:pseudoclase { propiedad: valor; }

Las pseudoclases más comunes son:

  • :hover: se activa cuando el ratón está sobre el elemento
  • :focus: se activa cuando el elemento tiene el foco
  • :active: se activa cuando el elemento está siendo activado (por ejemplo, al hacer clic)
  • :visited: se activa cuando un enlace ha sido visitado
  • :not(): se activa cuando un elemento no coincide con un selector
  • :nth-child(n): se activa en el enésimo hijo de un elemento

Pseudoelementos

Un pseudoelemento permite aplicar estilos a una parte específica de un elemento. La sintaxis es la siguiente:

CSS
selector::pseudoelemento { propiedad: valor; }

Los pseudoelementos más comunes son:

  • ::before: se utiliza para insertar contenido antes del contenido de un elemento
  • ::after: se utiliza para insertar contenido después del contenido de un elemento
  • ::first-line: se aplica al primer renglón de un bloque de texto
  • ::first-letter: se aplica a la primera letra de un bloque de texto

Variables

Las variables en CSS permiten almacenar valores reutilizables para propiedades de estilo. Se definen utilizando la sintaxis de --nombre-variable: valor; dentro de un selector, y se utilizan con la función var(--nombre-variable).

CSS
:root {
   --color-primario: #3498db;
   --margen-estandar: 16px;
}

.boton {
   background-color: var(--color-primario);
   margin: var(--margen-estandar);
}

Nota: Estas se guardan en root y comienzan siempre con doble guion

Fuentes

pasos para cargar una fuente externa:

  1. Elegir una fuente de un servicio de fuentes web (por ejemplo, Google Fonts).
  2. Incluir el enlace de la fuente en la sección <head> de tu HTML.
  3. Aplicar la fuente a los elementos deseados utilizando CSS.

Comentarios en CSS

Los comentarios de CSS consisten en señalar algo que se ignorará. Para establecer un comentario CSS se lo envuelve entre /* y */, independiente de la cantidad de líneas

CSS
/* Este es un comentario */

Nota: se puede usar control + / para comentar o descomentar líneas en muchos editores de código.

Propiedades con más uso

Propiedades de texto

  • color: Establece el color del texto. color: red;
  • font-size: Establece el tamaño de la fuente. font-size: 16px;
  • font-family: Establece la familia de fuentes. font-family: Arial, sans-serif;
  • font-weight: Establece el grosor de la fuente, con valores de 100 a 900. font-weight: 400;
  • font-style: Establece el estilo de la fuente. font-style: italic;
  • color: Establece el color del texto. color: red;
  • line-height: establece cómo será el interlineado de la fuente, es decir, el interlineado es la altura entre una línea y otra línea. line-height: 1.5;
  • text-align: Establece la alineación del texto. Puede tomar los valores left, right, center o justify. text-align: center;
  • letter-spacing: Establece el espacio entre las letras. letter-spacing: 2px;
  • word-spacing: Establece el espacio entre las palabras. word-spacing: 4px;
  • text-transform: Establece la transformación del texto (mayúsculas, minúsculas, etc.). Los valores que puede tomar son: none, capitalize, uppercase y lowercase. text-transform: uppercase;
  • text-decoration: Establece la decoración del texto (subrayado, tachado, etc.). Los valores que puede tomar son: none, underline, overline, line-through y blink. text-decoration: underline;
  • text-shadow: Establece la sombra del texto. Los valores que puede tomar son: [desplazamiento-x] [desplazamiento-y] [difuminado] [color]. text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);

Propiedades para elementos

  • width: Establece el ancho de un elemento. width: 100px;
  • height: Establece la altura de un elemento. height: 50px;
  • border: Establece el borde de un elemento. Los valores que puede tomar son: [grosor] [estilo] [color]. border: 1px solid #000;
  • border-radius: Establece el radio de las esquinas de un elemento. border-radius: 5px;
  • background-color: Establece el color de fondo de un elemento. background-color: #f0f0f0;
  • text-align: Establece la alineación del texto. Los valores que puede tomar son: left, right, center y justify. text-align: center;
  • vertical-align: Establece la alineación vertical de un elemento en línea o una celda de tabla. Los valores que puede tomar son: baseline, top, middle, bottom, text-top, text-bottom, sub y super. vertical-align: middle;
  • box-shadow: Establece la sombra de un elemento. Los valores que puede tomar son: [desplazamiento-x] [desplazamiento-y] [difuminado] [color]. box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);
  • display: Establece el tipo de visualización de un elemento. Los valores más comunes son: block, inline, inline-block, none y flex. display: none;

Propiedades para imagenes

  • float: Coloca la imagen flotando dentro del texto. Los valores que puede tomar son: left, right y none. float: left;
  • clear: Se coloca en el elemento de texto que no queremos que este flotando con la imagen. Los valores que puede tomar son: left, right, both y none. clear: both;
  • object-fit: Define cómo se ajusta una imagen dentro de su contenedor. Los valores más comunes son: "fill" para que la imagen llene todo el espacio disponible, "contain" para que la imagen se ajuste dentro del contenedor sin recortarse, "cover" para que la imagen cubra todo el contenedor, "none" para que la imagen no se ajuste y "scale-down" para que la imagen se reduzca si es más grande que el contenedor. object-fit: cover;
  • object-position: Define la posición de una imagen dentro de su contenedor cuando se utiliza object-fit. Los valores más comunes son: center, top, bottom, left, right y combinaciones como top left, bottom right, etc. object-position: center;

Propiedades para fondos de pantalla

  • background-image: Establece una imagen de fondo para un elemento. background-image: url('ruta/de/la/imagen.jpg');
  • background-size: Controla el tamaño de la imagen de fondo. Los valores más comunes son: cover, contain y auto. background-size: cover;
  • background-position: Define la posición de la imagen de fondo dentro del elemento. Los valores más comunes son: center, top, bottom, left, right y combinaciones como top left, bottom right, etc. background-position: center;
  • background-repeat: Controla si la imagen de fondo se repite o no. Los valores más comunes son: repeat, no-repeat, repeat-x, repeat-y, space y round. background-repeat: no-repeat;
  • background-attachment: Controla si la imagen de fondo se desplaza con el contenido o se fija en su lugar. Los valores más comunes son: scroll, fixed y local. background-attachment: fixed;

Propiedades para listas

  • list-style-type: Define el tipo de marcador para los elementos de una lista. Los valores más comunes son: disc, circle, square, decimal, lower-alpha, upper-alpha, lower-roman, upper-roman y none. El valor por defecto es disc. list-style-type: square;
  • list-style-position: Define la posición del marcador en relación con el contenido del elemento de la lista. Los valores son: inside (dentro del contenido) y outside (fuera del contenido). El valor outside es el valor por defecto. list-style-position: inside;
  • list-style-image: Permite utilizar una imagen como marcador para los elementos de una lista. Se especifica la URL de la imagen. list-style-image: url('ruta/de/la/imagen.png');

Transformaciones, Transiciones y Animaciones

Transformaciones en 2D

Las transformaciones en 2D permiten modificar la forma de los elementos en el espacio 2D. Las transformaciones más comunes son:

  • translate(x, y): Mueve un elemento en el eje X e Y. Los argumentos x e y representan la distancia a mover en píxeles. transform: translate(50px, 100px);
  • scale(sx, sy): Escala un elemento en el eje X e Y. Los argumentos tienen por defecto el valor 1, por lo que si colocamos el valor 1.2, va a aumentar el objeto en un 20%. transform: scale(1.5, 2);
  • rotate(angle): Rota un elemento alrededor de su centro en un ángulo especificado. El argumento angle se mide en grados. transform: rotate(45deg);
  • skew(ax, ay): Inclina un elemento en el eje X e Y. transform: skew(20deg, 10deg);

La propiedad transform-origin permite cambiar el punto de origen de las transformaciones. Por defecto, este punto es el centro del elemento. transform-origin: top left;

Transformaciones en 3D

Las transformaciones en 3D se usan en conjunto con las transformaciones en 2D, para usar las transformaciones vamos a tener contenedores que van a tener profundidad. Estas son:

  • translate3d(x, y, z): Mueve un elemento en los ejes X, Y y Z. transform: translate3d(50px, 100px, 30px);
  • scale3d(sx, sy, sz): Escala un elemento en los ejes X, Y y Z. transform: scale3d(1.5, 2, 1);
  • rotate3d(x, y, z, angle): Rota un elemento alrededor de un eje definido por los vectores x, y y z en un ángulo especificado. transform: rotate3d(1, 0, 0, 45deg);
  • perspective(n): Define la perspectiva para las transformaciones en 3D. El valor n representa la distancia desde el punto de vista hasta el plano de proyección. perspective: 500px;

Nota: Las transformaciones en 3D requieren un contexto de apilamiento de capas. Asegúrate de que el elemento padre tenga una propiedad perspective establecida para obtener el efecto 3D deseado.

Transiciones

Con las transiciones podemos tener las transformaciones que ya vimos, pero en lugar de que el cambio sea instantaneo ahora se va a tomar un tiempo el elemento. Esta se usa con un contenedor y un item dentro de este contenedor, cuando el item cambia de estado (por ejemplo, al hacer hover), la transición se activa:

CSS
.contenedor:hover .item {
   transform: rotateX(20deg);
}

Primero decimos que cuando estemos sobre el contenedor, el item va a rotar 20 grados.

CSS
.item {
   transition: 1.5s ease-in-out 0s;
}

Después le decimos al item que se demore 1.5 segundos en realizar esa transformacion.

Los argumentos de transition son los siguientes:

CSS
.item {
   transition: [transition-duration] [transition-timing-function] [transition-delay];
}
  • transition-duration: Define la duración de la transición.
  • transition-timing-function: Controla la velocidad de la transición a lo largo del tiempo (ease, linear, ease-in, ease-out, etc.).
  • transition-delay: Establece un retraso antes de que comience la transición.

Animaciones

Podemos hacer transiciones que no dependan de una accion por parte del usuario, sino que ocurran por un tiempo determinado, a estas se le llaman animaciones.

Para aplicar la animación a un elemento, se utiliza la propiedad animation, que puede desglosarse en las siguientes subpropiedades:

animation: [name] [duration] [timing-function] [delay] [iteration-count] [direction] [fill] [mode] [play-state]

Nota: solo las propiedades name y duration son obligatorias, las otras son opcionales.

  • name: Especifica el nombre de la animación definida con @keyframes.
  • duration: Define la duración de la animación.
  • timing-function: Controla la velocidad de la animación a lo largo del tiempo (ease, linear, ease-in, ease-out, etc.).
  • delay: Establece un retraso antes de que comience la animación.
  • iteration-count: Define cuántas veces se repetirá la animación (un número específico o infinite para repetir indefinidamente).
  • direction: Controla la dirección de la animación (normal, reverse, alternate, alternate-reverse).
  • fill: Controla el comportamiento de la animación cuando no está en ejecución (forwards, backwards, both, none).
  • play-state: Controla si la animación está en ejecución o pausada (running, paused).

Por otro lado, tenemos los keyframes, estas son las intrucciones de que va a pasar durante la duracion. Por ejemplo, hagamos que un ojo parpadee

CSS
.ojo {
   animation: parpadeo 3s infinite;
}

@keyframes parpadeo {
   0% { transform: scale(1, 1); }
   50% { transform: scale(1, 0.2); }
   100% { transform: scale(1, 1); }
}

En este ejemplo, al objeto con clase “ojo” le decimos que “parpadee”, y en el keyframe especificamos que es parpadear, siendo esto igual a que en el momento inicial se vea el ojo normal, en momento del medio la parte Y se reduzca hasta un 20% de su valor, y al final vuelva a verse el ojo normal

Tipos de display

Los tipos de display en CSS controlan cómo se muestra un elemento en la página. Los más comunes son:

  • block: El elemento ocupa todo el ancho disponible y comienza en una nueva línea.
  • inline: El elemento solo ocupa el espacio necesario y no comienza en una nueva línea. No es posible establecer un ancho o alto.
  • inline-block: Combina características de los elementos block e inline. Permite establecer un ancho y alto.
  • none: El elemento no se muestra en la página y no ocupa espacio.
  • flex: Permite un diseño flexible y adaptable en un solo eje.
  • grid: Permite un diseño basado en cuadrículas.

display flex

Propiedades para los items:

  • order: Define el orden de los elementos flexibles dentro del contenedor. El valor por defecto es 0. Los elementos con valores más bajos se muestran primero. order: 1;

Display Grid

El display grid es un modelo de diseño en CSS que permite crear diseños bidimensionales mediante la definición de filas y columnas en un contenedor. Al aplicar display: grid; a un contenedor, los elementos hijos se organizan automáticamente en una cuadrícula según las reglas definidas.

Propiedades para el contenedor:

  • grid-template-columns: Define el número y tamaño de las columnas en la cuadrícula. grid-template-columns: 100px 200px 1fr;
  • grid-auto-rows: Define el tamaño de las filas en la cuadrícula. Este es usado cuando se agregan más filas de las definidas en grid-template-rows. grid-auto-rows: 100px;
  • grid-template-rows: Define el número y tamaño de las filas en la cuadrícula. grid-template-rows: 100px 200px;
  • grid-auto-columns: Define el tamaño de las columnas en la cuadrícula. Este es usado cuando se agregan más columnas de las definidas en grid-template-columns. grid-auto-columns: 100px;
  • grid-gap: Controla el espacio entre filas y columnas en la cuadrícula. El valor por defecto es 0. Se pueden colocar dos valores, el primero para filas y el segundo para columnas. grid-gap: 10px 20px;
  • justify-items: Alinea los elementos dentro de sus celdas en el eje horizontal. Los valores comunes son start, end, center y stretch. El valor por defecto es stretch. justify-items: center;
  • align-items: Alinea los elementos dentro de sus celdas en el eje vertical. Los valores comunes son start, end, center y stretch. El valor por defecto es stretch. align-items: center;
  • grid-auto-flow: Controla cómo se colocan los elementos en la cuadrícula cuando no hay espacio definido. Los valores comunes son row, column y dense. El valor por defecto es row. grid-auto-flow: row;
  • grid-template-areas: Define áreas de cuadrícula con nombres para facilitar la colocación de elementos. Los nombres de área se asignan a los elementos hijos. grid-template-areas: "header header" "sidebar content"; Para dejar un area en blanco usamos un punto (.), por ejemplo grid-template-areas: "header header" "sidebar .";

Propiedades para los items:

  • grid-column: Define la posición y el tamaño de un elemento en las columnas de la cuadrícula. grid-column: 1 / 3;
  • grid-row: Define la posición y el tamaño de un elemento en las filas de la cuadrícula. grid-row: 2 / 4;
  • justify-self: Alinea un elemento dentro de su celda en el eje horizontal. Los valores comunes son start, end, center y stretch. El valor por defecto es stretch. justify-self: center;
  • align-self: Alinea un elemento dentro de su celda en el eje vertical. Los valores comunes son start, end, center y stretch. El valor por defecto es stretch. align-self: center;
  • grid-area: Asigna un elemento a un área de cuadrícula definida en grid-template-areas. grid-area: header;

Modelo de Caja

El modelo de caja se compone de cuatro elementos: margin, border, padding y contenido

Si entras a las herramientas de desarrollador de tu navegador y señalas un elemento HTML, en la sección de estilos te aparecerá una vista, este es el modelo de caja del elemento señalado. De adentro hacia afuera, los elementos son:

  • Content (Contenido): Es el área donde se muestra el contenido del elemento, como texto o imágenes.
    CSS
    div {
       width: 200px;
       height: 100px;
    }
  • Padding (Relleno): Es el espacio entre el contenido y el borde del elemento. Aumentar el padding incrementa el espacio alrededor del contenido.
    CSS
    div {
       padding: [arriba] [derecha] [abajo] [izquierda];
    }
  • Border (Borde): Es la línea que rodea el padding y el contenido. Puedes personalizar su grosor, estilo y color.
    CSS
    div {
       border: [grosor] [estilo] [color];
    }
  • Margin (Margen): Es el espacio fuera del borde del elemento. Aumentar el margen separa el elemento de otros elementos en la página.
    CSS
    div {
       margin: [arriba] [derecha] [abajo] [izquierda];
    }

Valores iniciales por defecto

Los valores iniciales por defecto del modelo de caja en CSS son:

CSS
* {
   margin: 0;
   padding: 0;
   box-sizing: border-box;
}

Propiedad box-sizing

La propiedad box-sizing establece cómo se calculará el width y el height sí incluyen bordes y espacios internos. Como buena práctica, se lo coloca en el selector universal, para que todos los elementos sigan esta tendencia. Los valores más comunes son:

  • content-box: El ancho y alto incluyen solo el contenido, sin padding ni border.
  • border-box: El ancho y alto incluyen el padding y el border, lo que facilita el diseño.

Overflow

La propiedad overflow controla cómo se maneja el contenido que desborda el área asignada a un elemento. Los valores más comunes son:

  • visible: El contenido desbordado es visible fuera del área del elemento (valor por defecto).
  • hidden: El contenido desbordado se oculta y no es visible.
  • scroll: Se agregan barras de desplazamiento para permitir al usuario desplazarse y ver el contenido desbordado.
  • auto: Se agregan barras de desplazamiento solo si es necesario.
  • overflow-y: Controla el desbordamiento en el eje vertical.
  • overflow-x: Controla el desbordamiento en el eje horizontal.

Posicionamiento en CSS

El posicionamiento en CSS consiste en cómo un elemento se situará, con respecto a su elemento padre y al flujo normal del documento. El flujo normal del documento es el orden de los elementos establecidos en el HTML. Las posiciones conocidas son:

  • Static (Estático): Es la posición por defecto de los elementos. Los elementos se colocan en el flujo normal del documento, uno después del otro.
  • Relative (Relativo): El elemento se posiciona relativamente a su posición original en el flujo del documento. Puedes usar las propiedades top, right, bottom y left para moverlo desde su posición original.
  • Absolute (Absoluto): El elemento se posiciona absolutamente con respecto a su contenedor posicionado más cercano (un elemento padre con posición relativa, absoluta o fija). Si no hay un contenedor posicionado, se posiciona con respecto al viewport.
  • Fixed (Fijo): El elemento se posiciona con respecto al viewport y permanece en la misma posición incluso cuando se hace scroll en la página.
  • Sticky (Pegajoso): El elemento se comporta como relativo hasta que alcanza un umbral definido por las propiedades top, right, bottom o left, momento en el cual se comporta como fijo.

Apilamiento

El apilamiento en CSS se refiere al orden en el que se muestran los elementos en la pantalla, especialmente cuando se superponen. Este orden se determina por la propiedad z-index, que establece la posición de un elemento en el eje Z (profundidad). Los elementos con un z-index más alto se muestran frente a los elementos con un z-index más bajo. Por defecto, todos los elementos tienen un z-index de 0.

Nota: si un elemento que tiene un z-index menor a otros, sus hijos nunca estarán por encima, aunque su z-index sea mayor

Medidas

En CSS, las unidades de medida se utilizan para definir tamaños, distancias y otros valores relacionados con el diseño y la presentación de los elementos en una página web. Las unidades de medida más comunes en CSS son:

  • Pixeles (px): Unidad absoluta que representa un punto en la pantalla. Es fija y no cambia con el tamaño de la pantalla o la resolución.
  • Porcentajes (%): Unidad relativa que representa un porcentaje del tamaño del elemento contenedor. Es útil para diseños fluidos y adaptativos.
  • Em (em): Unidad relativa que se basa en el tamaño de fuente del elemento padre. 1em es igual al tamaño de fuente del elemento padre.
  • Rem (rem): Unidad relativa que se basa en el tamaño de fuente del elemento raíz (generalmente el elemento html). 1rem es igual al tamaño de fuente del elemento raíz.
  • Viewport Width (vw): Unidad relativa que representa un porcentaje del ancho del viewport (área visible de la página web). 1vw es igual al 1% del ancho del viewport.
  • Viewport Height (vh): Unidad relativa que representa un porcentaje de la altura del viewport. 1vh es igual al 1% de la altura del viewport.

Configurar medida rem a 10px

Para configurar la medida rem a 10px, se debe establecer el tamaño de fuente del elemento raíz (html) a 62.5%. Esto se debe a que el tamaño de fuente por defecto en la mayoría de los navegadores es 16px, y al establecerlo en 62.5%, 1rem equivaldrá a 10px (16px * 0.625 = 10px).

CSS
html {
   font-size: 62.5%;
}

Responsive Design

El diseño responsivo (Responsive Design) es una técnica de diseño web que permite que una página web se adapte y se vea bien en diferentes dispositivos y tamaños de pantalla, como computadoras de escritorio, tabletas y teléfonos móviles. Esto se logra utilizando CSS y HTML de manera flexible y adaptable.

Media Queries

Las Media Queries son una característica de CSS que permite aplicar estilos específicos según las características del dispositivo o la ventana del navegador, como el ancho, la altura, la resolución y la orientación. Esto es fundamental para crear diseños responsivos que se adapten a diferentes tamaños de pantalla.

CSS
@media (max-width: 768px) {
   /* Estilos para pantallas con un ancho máximo de 768px */
}
